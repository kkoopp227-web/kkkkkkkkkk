# بوت رابط السيرفر الدائم (Permanent Invite Bot)

بوت بسيط مخصص لتزويد المستخدمين برابط دعوة دائم للسيرفر لا يتغير أبداً.

## الإعدادات السريعة (Quick Setup)
1. افتح ملف `config.json` وقم بتعبئة البيانات التالية:
    - `token`: توكن البوت الخاص بك.
    - `inviteLink`: ضع هنا رابط السيرفر الدائم (تأكد أن الرابط تم إنشاؤه ليكون "Set results to never expire").

2. التشغيل:
    - قم بتثبيت المكتبات: `npm install`
    - ابدأ التشغيل: `npm start`

## كيفية الاستخدام
- داخل السيرفر، أي شخص يكتب كلمة **"رابط"** أو **"رابطي"** في الشات، سيقوم البوت بالرد عليه فوراً برسالة تحتوي على الرابط الدائم بشكل احترافي.

تم التطوير لمنصة استضافة البوتات السحابية.
